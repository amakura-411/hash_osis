
**「＃推し（ハッシュタグ　推し）」**は、ユーザーにキャラクターとそのキャラについての特徴を登録してもらうサイトです。
　キャラを登録してもらうことで、ユーザーに今まで興味のなかった作品のキャラを知ってもらったり、あるいは自分が好きなキャラの特徴を理解する手助けになればよいと考えています。


「＃推し」の機能は以下の通りです
　＊ログイン必須
１.キャラクターの登録
２．キャラクターをお気に入りにする
　＊ログインなしでも可能
３．キャラクターを属性別に検索ができる
４．自分以外のユーザーがお気に入りにしたキャラクターの閲覧が可能
